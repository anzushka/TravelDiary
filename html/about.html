About

Japan Memory Map is a small interactive web project built with HTML, CSS and JavaScript.

The main feature is an SVG map of Japan. Each prefecture can be selected and connected to personal travel entries.

Entries are saved locally in the browser using LocalStorage. The library page gives an overview of all memories, while the stats page summarizes the travel progress.

Features

Interactive SVG map · DOM manipulation · Event listeners · LocalStorage · Editable travel entries · Dynamic statistics
